The Questions:

  • What is a CSS selector?

    A CSS selector is used to choose which HTML element you want to style.

  • What is the difference between an ID and a Class?

    An ID is used for one unique element, while a class can be used for many elements.

  • What is the difference between a SPAN tag and a DIV tag?

    A span is inline, while a div is a block element.

  • What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

    A div is a generic container, while header, main, and footer are semantic elements that provide meaning to the structure of the webpage.

  • What are the different ways to select a color?

    You can select a color using its name, hexadecimal code, RGB values, or HSL values.
